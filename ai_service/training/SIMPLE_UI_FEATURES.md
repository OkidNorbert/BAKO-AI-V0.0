# 🏀 Simple YOLOv8 Basketball Training UI

## ✅ **Essential Features Only**

### **🎯 What's Included (Focused on Your Project):**

#### **1. 📊 Dataset Statistics**
- **Total Frames Count** - See how many frames you have
- **Area Breakdown** - Ball, Player, Court Lines, Hoop frame counts
- **Real-time Updates** - Refresh stats after extraction

#### **2. 🎬 Video Upload & Frame Extraction**
- **Single Video Upload** - Simple file selection
- **Area-Specific Extraction** - Choose which basketball areas to extract
- **4 Basketball Areas**:
  - 🏀 **Ball** - Basketball object detection
  - 👤 **Player** - Basketball player detection
  - 📏 **Court Lines** - Court boundary detection
  - 🏀 **Hoop** - Basketball hoop detection
- **Configurable Limits** - Set max frames per area (10-200)

#### **3. 🏋️ YOLOv8 Training**
- **Essential Parameters**:
  - Epochs (10-200)
  - Batch Size (1-32)
  - Image Size (320, 416, 640)
  - Device (CPU/GPU)
- **One-Click Training** - Start with optimal settings

#### **4. 🧪 Model Testing**
- **Image Upload** - Test on any image
- **Confidence Threshold** - Adjust detection sensitivity
- **Quick Results** - See model performance

## 🚀 **How to Use:**

### **Step 1: Launch Simple UI**
```bash
cd /home/okidi6/Documents/Final-Year-Project/ai_service/training
source ../venv/bin/activate
python simple_yolo_ui.py
```

### **Step 2: Open Browser**
- Go to: `http://localhost:8003`
- Clean, simple interface
- Only essential features

### **Step 3: Workflow**
1. **Upload Video** → Select basketball video
2. **Choose Areas** → Select which areas to extract (ball, player, court, hoop)
3. **Extract Frames** → Get targeted frames for each area
4. **Start Training** → Train YOLOv8 with simple settings
5. **Test Model** → Test on new images
